Packt Video Subtitle Real-time Synchronizer Privacy Policy

Effective Date: November 21, 2025

1. Introduction

Packt Video Subtitle Real-time Synchronizer (hereinafter referred to as "this extension") is a Chrome browser extension created by the developer for real-time subtitle translation synchronization on Packtpub video pages (relying on the Immersive Translate extension).

We value your privacy. This privacy policy explains how this extension handles data. We do not collect any personally identifiable information (PII) or sensitive user data.

2. Data Collected

This extension only stores the following non-sensitive settings data locally, using the Chrome browser's chrome.storage.sync API (data is securely stored by Google and synced across devices, but not associated with your Google account):

  • Feature enable/disable status (boolean)
  • Subtitle font size (number, default 16)
  • Subtitle vertical offset (number, default 0)

These data are completely anonymous and used solely for the extension's functionality. They are not transmitted to any third-party servers.

3. Data Usage

Data is used only for:

  • Controlling whether the subtitle mapping feature is enabled.
  • Adjusting the display style of translated subtitles (font size, position).

This extension reads public DOM elements (such as video subtitle text) on Packtpub video pages (https://subscription.packtpub.com/video/*), performs local matching, and displays translation results (reusing the output from the Immersive Translate extension). It does not read, store, or transmit any video content, user input, or personal data.

4. Data Sharing

We do not share data with any third parties. chrome.storage.sync data is handled only by your browser and Google infrastructure.

5. Data Retention

Data is stored in your browser. You can clear it at any time via Chrome extension settings.

6. Your Rights

Access/Delete: Disable or remove the extension via chrome://extensions/.
Do Not Track: This extension does not track browsing behavior.

7. Children's Privacy

This extension is suitable for users of all ages, including children under 13. The extension's content is healthy and safe, with no harmful information.

8. Changes Notification

Major changes will be notified through extension updates.
